# Landsat Web Tracker

Project made by EKK Ingarden for [Nasa Space Apps 2024](https://www.spaceappschallenge.org/).

## Prerequisites
- Node.js
- Pnpm
- Poetry

## Installation
```bash
pnpm install
```

## Running
```bash
# Frontend
pnpm frontend:dev

# Backend
pnpm backend:dev

# Everything
pnpm dev
```
